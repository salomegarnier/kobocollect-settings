Configure additional devices

Once you have configured these settings for one device, you can generate a QR code to copy the same server settings and project configuration to other devices.

Generate the QR code from the configured device

  1. Open the Projects menu and select the project you want to copy.
  2. Tap Settings.
  3. Select Project Management.
  4. Tap Reconfigure with QR Code.
  5. Choose QR Code. The QR code will appear on the screen.
  6. Take a screenshot of the QR code, or return to this menu later to access it again.

Configure another device with the QR code

  1. Open KoboCollect on the device you want to configure.
  2. Tap Configure with QR Code.
  3. Scan the QR code with the device camera, or tap the three dots icon in the top right corner and select Import QR code to use a saved screenshot.
Important: The QR code contains the account credentials and access permissions for the account used to generate it. To configure enumerator devices, generate the QR code from an account that only has the permissions enumerators need.
